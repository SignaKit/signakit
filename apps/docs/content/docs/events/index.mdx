# SignaKit Events

<Callout type="info" title="Coming soon">
  SignaKit Events documentation is under development. The Events SDKs for Node.js, browser, and React are in progress.
</Callout>

Custom event tracking and analytics for web and mobile. SignaKit Events lets you track any user action — button clicks, purchases, form submissions — with flexible properties and server-side or client-side SDKs.

**Planned SDKs:**

- `@signakit/events-node` — server-side event tracking
- `@signakit/events-browser` — client-side event tracking
- `@signakit/events-react` — React wrapper

Want to be notified when Events docs are ready? Watch the [changelog](/flags/changelog) for updates.
